# SIMPLE HTTP CLIENT

## General setup

(Note that the `$` denotes a command-line prompt to be typed in a system terminal, do not type it)

1. Make sure you have installed NodeJS in your system: https://nodejs.org
   
2. Open a terminal in this folder.

3. Make sure Node is available by typing: 
   
    `$ node --version`

4. Install the dependencies of this project:
   
    `$ npm install`

5. Make sure you have the `http-server-node` running on your machine. 


## Init

1. Open a terminal in this folder, and make a `GET` request to the server:
   
    `$ node get-request.js`

2. Make a `POST` request to the server:
   
    `$ node post-request.js`
